# Security Policy

Please do not disclose vulnerabilities in a public issue.

Report a security concern to `info@carolina3dprintstudio.com` with:

- the affected worker version or commit;
- a concise description and reproduction steps;
- expected impact;
- whether client data or credentials may be involved.

Do not include client models, credentials, access tokens, or decrypted production artifacts in the report. Receipt will be acknowledged as soon as practical.
